Introduction

In this program:

  • It creates a dynamic datagrid
  • It creates headers dynamically
  • Opens a file and reads it line by line
  • It fills the datagrid dynamically according to the text file you use
  • You can set any separator you like for the text files to split into columns
  • It skips empty lines
  • This sample also works if you have less or more data than your column header

With this program you can open any .txt or .csv file with any divider and place it into a dynamicaly added datagridview

The Datagridview is completly created at runtime! It does not matter how many columns or lines your text file has, it is all dynamicaly adjusted!

The program itself is less than 75 lines so very easy to ajust to your own purpose 

The datagridview is added at runtime and columns are dynamicaly placed according to the number of columns you have in your txt files you can easely change the seperator e.g. to ";", "^,^" or whatever you like

Improved version first checks the number of columns if there are no headers. If you use the first row as columns it already worked that way. Also another improvement, The program skips empty lines so no error occures when you have an extra return at the end

Building the Sample

You do not need any extra's for this sample, just download and run

The sample creates a text file ";" seperated if none exist

If you have your own text file or csv file, you can just adjust the path for the variable "FILE_NAME"

If your text file containes less headers than data columns an error will be displayed at lblerror.text. To see the full error just check the debug window
